鐵日誌 IRONLOG — 自架版上架步驟（GitHub Pages，免費、永久、不綁 Claude 帳號）
================================================================

這個資料夾共 5 個檔案：
  index.html            App 本體（單檔，含全部功能）
  sw.js                 離線支援（service worker）
  manifest.webmanifest  App 設定（名稱、圖示、全螢幕）
  icon-180.png          iPhone 主畫面圖示
  icon-512.png          通用圖示

一次性設定（用電腦操作，約 5–10 分鐘）：

1. 到 github.com 用「私人 email」註冊帳號（已有帳號就直接登入）
2. 右上角「+」→「New repository」
   - Repository name 填：ironlog
   - 選 Public（程式碼會公開，但你的訓練資料「不會」在裡面——
     資料只存在你手機的瀏覽器）
   - 按「Create repository」
3. 頁面中間點「uploading an existing file」
   → 把這 5 個檔案全部拖進去 →「Commit changes」
4. 上方「Settings」→ 左邊「Pages」
   → Branch 選「main」→「Save」
5. 等 1–2 分鐘，重新整理 Pages 頁面，會出現你的專屬網址：
   https://你的帳號.github.io/ironlog/

iPhone 設定（30 秒）：
6. 用 Safari 打開上面的網址
7. 點下方「分享」→「加入主畫面」
8. 完成！主畫面會出現藍色槓鈴圖示，點開是全螢幕 App，
   之後沒網路也能開（第一次打開後即離線可用）

搬資料（如果你在舊的 Claude 連結已經有紀錄）：
- 舊的：進度頁最下面「匯出」→ 複製 JSON
- 新的：進度頁「匯入」→ 貼上 → 完成

日常備份建議：每一兩個月到進度頁「匯出」一次，把 JSON 傳給自己存著。

之後想改功能：把 index.html 給任何 Claude 對話說要改什麼，
改完把新的 index.html 上傳到同一個 GitHub repo 覆蓋即可（網址不變）。
